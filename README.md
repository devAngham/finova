# Finova — AI-Powered Banking Assistant

> A production-ready AI Agent banking API that allows users to manage their finances through natural language conversation — powered by NestJS, TypeScript, PostgreSQL, Redis, WebSocket, and Groq LLaMA with MCP Tool Use.

---

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| Runtime | Node.js 18 |
| Language | TypeScript |
| Framework | NestJS |
| Database | PostgreSQL |
| ORM | TypeORM |
| Cache | Redis |
| Real-time | WebSocket |
| AI Model | Groq LLaMA 3.3 70B |
| AI Protocol | MCP Tool Use |
| Auth | JWT + Refresh Token |
| DevOps | Docker |
